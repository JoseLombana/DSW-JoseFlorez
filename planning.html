Selectores CSS

Cada regla o estilos esta compuesta por << selectores >> + { + << declaracion >>} El selector o selectores es el indicador del elemento o elementos HTML a los que se le aplica la regla CSS.
Por consiguiente otro termino que debemos mensionar que esta estrechamnete vinculado con los selectores es la declaracion.
La declaracion determina los estilos que se aplican a los elementos. Esta declaracion puede estar compuesta por una o mas propiedades CSS.
Una propiedad es una caracteristica que se modifica en el elemento seleccionado, ejemplo el tamaño de letra, color de fondo, etc.
A su vez otro termino es valor, este establece el nuevo de la caracteristica modificada en el elemento.

    Selectores CSS

    Cada regla o estilo esta compuesta por << selectores >> + {+ << declaracion >>} El selecotr o selectores es el indicador del elemento o elementos HTML a los que se le aplica la regla CSS.
    Por consiguiente otro termino que debemos mensionar que esta estrechamente vinulado con los selectores es la declaracion.
    La declaración determina los estilos que se aplican a los elementos. Esta delaracion puede estar compuesta por una o mas propiedades CSS.
    Una propiedad es una caracteristica que se modifica en el elemento seleccionado, ejemplo el tamaño de letra, color de fondo, etc.<7br> A su vez otro termino es valor, este establece el nuevo de la caracteristica modificada en el elemento.
